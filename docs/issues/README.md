# GitHub Issues for ATOMS3R Migration

このディレクトリには、ATOMS3R AI Chatbot Kit への移植プロジェクトのために作成すべき GitHub Issues のテンプレートが含まれています。

## 使い方

各ファイル（`issue-01.md` から `issue-12.md`）の内容をコピーして、GitHub のイシュー作成画面に貼り付けてください。

## メインイシュー

まず `issue-00-main.md` をメインイシューとして作成し、その後サブイシューを作成することをお勧めします。

## ラベル

以下のラベルを事前に作成しておくことをお勧めします：

- `atoms3r-migration` - ATOMS3R移植関連
- `enhancement` - 機能追加
- `high-priority` - 高優先度
- `medium-priority` - 中優先度
- `low-priority` - 低優先度
- `phase-1` - フェーズ1（基本移植）
- `phase-2` - フェーズ2（最適化）
- `phase-3` - フェーズ3（テスト・ドキュメント）
- `phase-4` - フェーズ4（将来の拡張）
