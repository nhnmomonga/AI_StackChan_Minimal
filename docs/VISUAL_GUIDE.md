## よくある質問（FAQ）

### Q1: なぜ ATOMS3R に移植するのですか？
**A:** ATOMS3R は以下の利点があります：
- より強力な ESP32-S3 プロセッサ
- 8MB PSRAM で大きなモデルに対応
- 内蔵カラーディスプレイで組み立て不要
- センサー内蔵で将来の拡張が容易

### Q2: ATOM Echo のサポートは続きますか？
**A:** はい。可能な限り後方互換性を維持します。条件コンパイルにより両方をサポート予定です。

### Q3: どちらを購入すべきですか？
**A:** 
- **初心者・簡単に始めたい**: ATOMS3R（組み立て不要）
- **コスト重視・DIY好き**: ATOM Echo（3,000円台）
- **最新機能を試したい**: ATOMS3R（PSRAM、センサー）

### Q4: 移植にはどれくらい時間がかかりますか？
**A:** 
- 基本機能のみ: 2-3週間
- すべての機能: 3-5週間
- （実機テストの時間を含む）

### Q5: 貢献するにはどうすればよいですか？
**A:** 
1. このリポジトリをフォーク
2. GitHub Issues から作業したいイシューを選択
3. 実装してプルリクエスト作成
4. レビュー後にマージ

## 用語集

| 用語 | 説明 |
|-----|------|
| **ATOM Echo** | M5Stack の小型マイコン（ESP32-PICO）、オーディオ機能付き |
| **ATOMS3R** | M5Stack の最新小型マイコン（ESP32-S3）、ディスプレイ・オーディオ・センサー統合 |
| **ESP32-S3** | Espressif の最新マイコンチップ、PSRAM対応 |
| **PSRAM** | 外部RAMメモリ、大容量データの処理に使用 |
| **SSD1306** | OLEDディスプレイドライバIC（モノクロ、I2C） |
| **GC9107** | TFTディスプレイドライバIC（カラー、SPI） |
| **M5Unified** | M5Stack デバイス用の統合ライブラリ |
| **PlatformIO** | 組み込み開発用のIDE/ビルドシステム |
| **I2C** | シリアル通信プロトコル（SDA/SCL） |
| **SPI** | シリアル通信プロトコル（高速、ディスプレイ等に使用） |
