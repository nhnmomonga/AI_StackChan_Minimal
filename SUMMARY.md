## 主要な変更点（予定）

### ハードウェア
| 項目 | ATOM Echo | ATOMS3R |
|------|-----------|---------|
| CPU | ESP32-PICO | ESP32-S3 @ 240MHz |
| メモリ | 限定的 | 8MB Flash + 8MB PSRAM |
| ディスプレイ | 外部SSD1306 (96x64) | 内蔵TFT (128x128) |
| I2C SDA | GPIO21 | GPIO38 |
| I2C SCL | GPIO25 | GPIO39 |

### ソフトウェア
1. **platformio.ini**
   - ボード設定を ESP32-S3 に変更
   - PSRAM を有効化

2. **src/main.cpp**
   - ピン定義の更新
   - ディスプレイ初期化の変更
   - メモリバッファの拡大

3. **条件コンパイル（オプション）**
   ```cpp
   #ifdef ATOMS3R
     // ATOMS3R 用コード
   #else
     // ATOM Echo 用コード
   #endif
   ```
